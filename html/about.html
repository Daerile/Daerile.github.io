About this website

This website was created to help those that got stuck in the world of Tekken. We make guides about characters which include the optimal comboes, most useful moves, their properties and everything you need to know about a character to pick them up. Learning a character in Tekken is hard, some consider it too hard. In the future, this website will contain useful tips not only limited to characters, but to the whole game aswell.
